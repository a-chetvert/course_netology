# Задача 1. Оценка алгосложности программы
Требуется определить алгосложность программы (код ниже). 

 ## Вопрос: что делает программа:
 Вычисляет разницу между первым и последним элементом массива.

 ## Вопрос: какая асимптотика:
 **По времени:** O(n), т.к. массив выполнится число **size** раз.

 **По памяти:** O(1), т.к. используется только одна дополнительная переменная **ans**.


```cpp
int calc(int* arr, int size)
{
  int ans = 0;
  for (int i = 1; i < size; i++)
  {
    ans += arr[i] - arr[i-1];
  }
  return ans;
}
```
